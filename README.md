# Usage
This project adds some new methods to the core ruby classes

To use core-ext either intall directly

```shell
gem install darthjee-core_ext
```

or add it to Gemfile

```shell
gem 'darthjee-core_ext'
```

```shell
bundle install darthjee-core_ext
```
